# Анализ данных развивающегося маркетплейса
## Цель исследования
Провести анализ данных маркетплейса, выявить прибыльные каналы и категории товаров.

## Задачи исследования
* Первичный анализ и сбор данных.
* Обзор ключевых метрик.
* Оценка метрик монетизации и юнит-экономики.
* Поиск инсайтов и точек роста. Сегментация и формулирование гипотез.
* Подготовка эксперимента и подведение его итогов.

  ## Стек
  * Jupyter Notebook, Python
 
  ## Выводы
- **Этапы оттока:** 37% посетителей теряются до оформления заказа.
- **Каналы привлечения:** самый перспективный — TikTok.
- **Прибыльные категории:** мебель для дома, бытовая техника, украшения и часы, спортивный инвентарь, товары для туризма.
- **Стратегия бюджета:** перераспределить средства в пользу TikTok.
- **A/B-тест (бонус на первую покупку):** выявлено статистически значимое повышение конверсии, но обнаружен эффект новизны — тест продлён для подтверждения.
